# Manual de usuario

Este manual describe el uso básico de la plataforma de bienestar y gimnasio de la Universidad Cali para los distintos roles disponibles. La aplicación cuenta con una interfaz web responsiva basada en React y se accede a través de un navegador moderno.

## Acceso a la aplicación

1. Abra su navegador y diríjase a la URL proporcionada por la Universidad.
2. Ingrese su **usuario institucional** y contraseña en la pantalla de inicio de sesión.
3. Tras autenticarse correctamente, el sistema le mostrará el panel correspondiente a su rol.

## Roles y funcionalidades

### Estudiante / Colaborador

* **Explorar ejercicios**: desde la sección *Ejercicios* puede ver una lista de ejercicios predefinidos. Puede filtrar por tipo (cardio, fuerza, movilidad) o nivel de dificultad. Si necesita un ejercicio que no existe, solicite a su entrenador que lo cree.
* **Crear rutinas**: en la sección *Rutinas* pulse *Nueva rutina* para crear una rutina personalizada. Asigne un nombre, descripción y seleccione ejercicios de la lista. Para cada ejercicio defina orden, series, repeticiones, duración o descanso según corresponda.
* **Adoptar rutinas prediseñadas**: consulte la lista de rutinas prediseñadas y utilice la opción *Adoptar*. Se generará una copia personal que podrá modificar.
* **Registrar progreso**: en la vista de una rutina seleccione *Registrar progreso* para añadir un registro diario o semanal. Indique ejercicio, repeticiones, tiempo y nivel de esfuerzo. Los registros aparecen en su historial de progreso.
* **Consultar estadísticas personales**: el panel muestra gráficas de su progreso a lo largo del tiempo y comparativas entre tipos de ejercicio.

### Entrenador

* **Gestionar ejercicios**: puede crear, editar o eliminar ejercicios desde la sección *Ejercicios*. Ingrese nombre, tipo, descripción, duración, nivel de dificultad y enlaces a videos demostrativos.
* **Crear rutinas prediseñadas**: utilice la opción *Crear rutina* marcando el campo *Rutina prediseñada*. Estas rutinas estarán disponibles para que los estudiantes las adopten.
* **Ver rutinas y progreso de usuarios**: acceda al módulo *Usuarios asignados* para consultar las rutinas y registros de progreso de cada estudiante o colaborador. Puede ordenar por fecha o por nivel de esfuerzo.
* **Enviar recomendaciones**: dentro del perfil de un usuario utilice el botón *Añadir recomendación* para registrar consejos personalizados. El usuario podrá verlas en su panel.
* **Estadísticas de desempeño**: el panel del entrenador incluye gráficos de número de asignaciones nuevas y seguimientos realizados por mes.

### Administrador (Jefe de Bienestar)

* **Asignar entrenadores**: desde el módulo *Administración* seleccione un usuario y asigne un entrenador de la lista disponible. Si existe una asignación previa, se cerrará automáticamente y se registrará la nueva.
* **Reasignar entrenadores**: siga el mismo procedimiento para actualizar la relación usuario–entrenador. Las asignaciones anteriores se marcan con fecha de finalización.
* **Ver informes globales**: acceda al panel de estadísticas para visualizar informes agregados, como ranking de constancia, ejercicios más populares y análisis de cumplimiento de objetivos.

## Buenas prácticas de uso

* Ingrese datos completos y verídicos en sus registros de progreso para obtener reportes más precisos.
* Los entrenadores deben revisar periódicamente los registros de sus usuarios para proporcionar retroalimentación oportuna.
* Los administradores deben mantener actualizadas las asignaciones de entrenadores para garantizar un acompañamiento adecuado.

## Soporte

Si presenta inconvenientes técnicos con la plataforma o desea sugerir mejoras, por favor comuníquese con el departamento de Sistemas Inteligentes a través del correo de soporte institucional.
